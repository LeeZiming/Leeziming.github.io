Github Page 搭建教程（一）

发表于 2019-06-08 |

​ 端午节在半小时前刚刚过去，祝大家端午节快乐！2019年的三天小长假也随之走过大半。今天正值高考之时，祝广大学子能够正常发挥，金榜题名。

​ 今天开始写我的第一篇教程，教大家如何利用GitHub Page建立自己的个人主页。

​ 互联网从诞生发展至今日，每天有无数的人在上面留下自己的足迹，他们用它记录生活、抒发情感、分享见解，这一切都离不开一个可以承载文字的平台，一个独立的、由自己掌控的平台，而 GitHub Pages 就是这么一个平台。

​ 在这个平台里你可以使用自己的个性域名；可以在海量的主题里挑选最适合你的那一款，如果你技术极客，也可以根据自己的喜好，设计属于自己的个性化页面；你既可以在线创建和发布网站，也可以在本地通过客户端工具或者命令行进行网站和内容的管理。

​ 你完全可以通过 GitHub Pages 展示和输出自身价值，甚至可以把它打造成属于自己的互联网「身份证」。

如何创建GitHub Page

​ 首先你需要注册一个GitHub账号，注册过程就不赘述了。

​ GitHub

创建域名

​ 进入你的首页后，新建一个Repository。

​ 在Respository name上按username.github.io的格式填写域名。

创建页面

创建成功后，在菜单栏点击右上角的Settings。

​ 找到GitHub Page选项，从官方提供的主题中选择一个主题。

​ 选择一个喜欢的风格，点击Select theme，系统就会帮你生成你的页面，点击提交，你的GitHub page就建好了。

​ 在浏览器输入你的项目名称，比如leonvirgo.github.io，就可以看到你刚刚选择的主题对应的个人主页了。

修改内容

到这里，如果你只是想做一个例如可以随时在互联网上访问的简历，那么你只需要 GitHub Pages 项目的主页修改你 index.md 文件就可以了，比如我给出的这个模板index.md.

​ 如果你不满足于此，进阶版内容下期更新～
